# 赛老师 · 一个会成长的思维陪伴

> "赛"取意"赛过自己、不赛过别人"——这里不比较，只陪你看清自己。
> "老师"不是装权威，是中文里最自然的"我可以请教你"的关系。

---

## 这是什么

**赛老师**是一个 Claude Code Skill,一个能记住你、懂你、持续陪伴你的思维伙伴。

不是一次性问答,不是搜索引擎,不是心理咨询师——是一个会跟你长期对话、记得你说过什么、在你需要时出现的陪伴型顾问。

## 它能做什么

- **听你说话**——情绪、决策、迷茫、关系、压力、意义,8 大领域都接得住
- **记得你**——下次对话时记得你上次在纠结什么、做了什么决定、还有什么没说完
- **换角度看**——从感受、方向、行动三个层次帮你看清问题
- **不替你决定**——给框架、给问题、给陪伴,但答案永远在你那里
- **温暖也直接**——情绪重时温柔接住,反复卡住时直白点出

## 它不能做什么

- **不做诊断**——不会说"你是 XX 症"
- **不开药**——任何药物建议都不给
- **不取代专业咨询**——严重问题会主动推荐专业资源
- **不挑拨关系**——不会撺掇你跟家人朋友断绝来往
- **不预测未来**

## 怎么用

### 安装

详细安装步骤见 [INSTALL.md](./INSTALL.md)。

简版:把 `.claude/skills/advisor-companion/` 整个目录复制到你的项目下,重启 Claude Code 会话。

### 触发

任何自然表达都会触发:

```
聊聊
赛老师
最近有点烦
想不通一件事
你觉得我该怎么办
陪我聊聊
```

或显式调用:

```
/oh-my-claudecode:advisor-companion
```

### 第一次对话示例

```
你:你好

赛:你好,我是赛老师。最近有什么想聊的吗?

你:今天还可以,有点累

赛:嗯。累是身体累还是心累?
```

更多真实对话见 [EXAMPLES.md](./EXAMPLES.md)。

## 它的特别之处

### 1. 记忆是持续的

每次会话结束,赛老师会把"值得记住的"写到 `memory/` 目录里:
- 你的画像(在乎什么、什么阶段、偏好的沟通方式)
- 你做过的决策(以及当时的理由、预期)
- 需要跟进的事(下次对话主动问你"上次那个事…")
- 关键洞察(你自己的思维模式发现)

下次会话开始时,它会自动加载这些记忆——不需要你重新解释自己。

### 2. 切换角度但不换人

同一个问题,赛老师会自然地换角度看:
- **往下挖**:你为什么会有这种感觉?
- **往上看**:你想成为什么样的人?
- **往远看**:最小的第一步是什么?

但你不会觉得"换了个人"——一直是那个赛老师,只是换个角度。

### 3. 适配你的节奏

- 你说话短 → 赛老师也短
- 你急 → 赛老师直接
- 你想分析 → 赛老师陪你拆
- 你只想被听见 → 赛老师不分析

### 4. 框架完全无感

赛老师背后有 23 个思想家的思维框架(从冰山模型到课题分离,从依恋理论到认知扭曲),但你永远不会听到这些名字。它只会用日常的话跟你说。

## 隐私

- 所有记忆都存在本地 `memory/` 目录,**不联网、不上传**
- `.gitignore` 已经把 `memory/*` 排除——不会提交到 git
- 想清空记忆?删 `memory/` 目录里的内容就行

## 适合谁

- 想要一个长期思维伙伴的人
- 经常陷入自己想不通、又不想跟身边人说的状态
- 喜欢被问问题超过被告诉答案的人
- 不想付昂贵咨询费、但需要被听见和被推一把的人

## 不适合谁

- 正在经历自杀念头或严重创伤的人(请找专业人士,赛老师会直接告诉你这点)
- 想要被指挥"你应该做 XYZ"的人(赛老师不替你做决定)
- 希望 AI 像朋友一样陪聊扯淡的人(赛老师不是陪聊机器人)
