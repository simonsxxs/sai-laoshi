# 伊瓦拉（Herminia Ibarra）完整时间线

> 根据 Wikipedia、London Business School 官方档案、Thinkers50 数据、INSEAD 档案、HBR 作者档案、ORCID 等来源交叉验证

## 早年与教育

| 年份 | 事件 |
|------|------|
| 约1961 | 出生于古巴，父母为古巴人，幼年随家庭移居美国（迈阿密） |
| 约1983 | 获得迈阿密大学（University of Miami）心理学学士学位 |
| 约1985 | 获得耶鲁大学（Yale University）硕士学位，组织行为学方向 |
| 1986-1989 | 在耶鲁大学攻读博士学位期间担任教学和助理研究员，师从组织行为学领域的教授 |
| 1989 | 获得耶鲁大学博士学位（组织行为学）——博士论文研究方向：组织社会化与职业适应 |

## 学术生涯

| 年份 | 事件 |
|------|------|
| 1989 | 加入哈佛大学肯尼迪政府学院（Harvard Kennedy School），任助理教授——开始研究生涯 |
| 1990s | 早期研究聚焦组织社会化（organizational socialization）和导师关系（mentoring relationships） |
| 1994 | 晋升哈佛大学副教授 |
| 1998 | 晋升哈佛大学正教授（获得终身教职 Tenure） |
| 1999 | 发表标志性论文 "Provisional Selves: Experimenting with Image and Identity in Professional Adaptation"（Administrative Science Quarterly, 44:4）——提出"临时的自我"概念，奠定她后续研究的理论基础 |
| 2000 | 发表 HBR 文章 "How to Get the Mentoring You Need"——早期大众影响力尝试，探讨如何主动建立导师关系 |
| 2000s | 与 Doug Hall 合著 Protean Career（无边界/多变的职业）相关论文——将自己的"行动优先"观点与 Hall 的"无边界职业"概念结合 |
| 2002 | 离开哈佛（放弃终身教职），加入 INSEAD（法国枫丹白露校区），任组织行为学教授——这是她职业生涯中最重要的"行动优先"决策 |
| 2002 | 发表学术论文 "How Do You Know Who You Are?"（Academy of Management Journal）——深化身份转变研究 |
| 2003 | 出版《Working Identity: Unconventional Strategies for Reinventing Your Career》——基于 39 位职业转型者的深度访谈，提出"行动先于认知"的核心论点 |
| 2003-2005 | 在 INSEAD 开展高管教育项目，将研究发现应用于企业领导力发展实践 |
| 2004 | 《Working Identity》被《Fast Company》杂志报道，引发商业界关注 |
| 2005 | 在多个国际会议上发表主题演讲（AOM Academy of Management, EGOS 等）——职业身份转型成为组织行为学新兴热点 |
| 2006-2010 | 深入 INSEAD 高管教育，将教学转化为研究素材——"学者也是实践者"思维 |
| 2008 | 发表第二篇重要 HBR 文章 "How to Get the Mentoring You Need"（重新编辑版），进一步提升大众知名度 |
| 2009 | 在加拿大、欧洲、美国的组织行为学会议上继续推进职业身份研究 |
| 2010 | Working Identity 成为职业转型领域的经典参考书，被多所商学院纳入课程 |
| 2011 | 首次入选 Thinkers50 榜单——标志着从纯学术到思想领袖的跨越 |
| 2012 | 在 INSEAD 创建领导力发展高管教育项目，将 Outsight 原则系统化 |
| 2012 | 在英国和美国多个行业组织（如 NHS、银行、咨询公司）运用她的框架进行领导力培训 |
| 2013 | 《Working Identity》更新版平装本出版，增加新序言讨论金融危机后的职业转型趋势和数字时代的副业文化 |
| 2013 | 发表 "Identity Transitions"（与 Sarah Wittman 和 Laura Morgan Roberts 合著）——收录于 Oxford Handbook of Identity |
| 2014 | 发表综述性论文 "Managerial and Professional Identity"（Annual Review of Organizational Psychology）——总结组织身份研究的 20 年成果 |
| 2015 | 出版《Act Like a Leader, Think Like a Leader》——将 Outsight 原则从职业转型扩展至领导力发展 |
| 2015 | 发表 HBR 文章 "The Authenticity Paradox"——成为年度最受关注的管理文章之一，引发广泛讨论 |
| 2015 | 发表 HBR 文章 "What to Do When Your Career Stalls"——Working Identity 理念的大众化浓缩版本 |
| 2015 | 被 Thinkers50 授予组织行为领域领导力奖（Leadership Award） |
| 2015 | TEDxLondon 演讲 "The Authenticity Paradox"——视频在多个平台传播，至今仍是管理领域播放量最大的 TEDx 之一 |
| 2015 | 发表 HBR 文章 "The Leadership Development Rut"——批判传统领导力培训的无效性 |
| 2015 | 被《金融时报》列为"管理思想领袖" |
| 2016 | 持续为 HBR 撰写多篇关于领导力转型的文章 |
| 2017 | 离开 INSEAD（结束 15 年任期），加入伦敦商学院（London Business School），任 Charles Handy 组织行为学讲席教授 |
| 2017 | The Case Centre 案例作者前 40 名 |
| 2018 | Thinkers50 排名第 4 位——职业生涯最高排名，仅次于几位管理学的标志性人物 |
| 2018 | 发表合著论文 "Making Sense of Who We Are"（与 Steven Hitlin 和 Nick Putnam）——Handbook of Social Psychology |
| 2018 | 在 Academy of Management 年会上主持关于"职业身份"的主题论坛 |
| 2019 | 当选英国社会科学院院士（Fellow of the British Academy, FBA）——英国社会科学领域的最高学术荣誉之一 |
| 2019 | 被 Poets & Quants 评为"最具影响力的商学院教授"之一 |
| 2020 | Thinkers50 排名第 8 位 |
| 2020 | 在 COVID-19 疫情期间参与多场关于"远程工作与职业身份"的线上讨论和 HBR 网络研讨会 |
| 2020 | 为多家企业在疫情后的组织转型提供咨询——领导力发展的新挑战 |
| 2020 | 接受 BBC 和 Financial Times 关于"疫情如何改变职业生涯"的专访 |
| 2021 | 《Act Like a Leader, Think Like a Leader》第二版出版——增加数字化转型、疫情后领导力、远程团队管理等新内容 |
| 2021 | 接受 Strategy+Business 深度访谈，讨论第二版新增内容和疫情对领导力的长期影响 |
| 2021 | 在 HBR 的 Ideacast 节目中与 Sarah Green 对谈，讨论"疫情后的领导力真实性" |
| 2022 | 接受多家媒体关于"大辞职潮"（The Great Resignation）的深入专访——她 20 年前的职业转型框架在大规模职业流动背景下被重新发现和热议 |
| 2022 | 参与 INSEAD 校友讲座系列，回顾 20 年学术生涯的关键转折 |
| 2022 | 被《The Times》和《The Guardian》引用关于"职业重塑"的评论 |
| 2023 | 发表 HBR 封面文章 "How to Reclaim Your Time"——探讨现代职业人士的时间被"日程殖民化"的困境，成为年度阅读量最高的 HBR 文章之一 |
| 2023 | 发表 HBR 文章 "Stop Self-Helping and Start Doing"（与另一作者合著）——继续推进"行动优先"的对话，批判过度自我反思文化 |
| 2023 | 在伦敦商学院开设新课程，将"时间管理"与"领导力发展"结合 |
| 2023 | 在多个亚洲国家（新加坡、印度）的商业论坛上发表关于职业转型的演讲——理论的亚洲应用探索 |
| 2024 | 继续担任伦敦商学院教授及理事（Governor） |
| 2024 | 参与 FBA 院士活动，推动社会科学在政策层面的影响 |
| 2024 | 在多个国际管理会议上发表演讲（AOM, SMS 等） |
| 2025 | 持续活跃于 Thinkers50 榜单和全球企业领导力发展实践 |
| 2025 | 在播客和商学院访谈中讨论"AI 时代下的职业身份重塑"——其 20 年前的理论在新的技术变革语境中产生新的共鸣 |
| 2026 | 担任 LBS 教授，持续研究和写作；其职业转型理论在"AI 时代职业重塑"的新语境中被重新热议；继续为全球企业提供领导力发展咨询 |
